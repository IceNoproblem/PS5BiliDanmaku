## 🎯 PS5配置

### 推流设置
1. 在 PS5 上选择"开始播放/游戏直播"
2. 推流服务选择 Twitch
3. 服务器地址填写：`你的电脑IP`（如 192.168.1.100）
4. 流密钥随意填写
5. 选择要转发的频道（如 #icenoproblem）

### IRC连接配置
1. 修改 PS5 的 DNS 设置为你的电脑 IP
2. 在 hosts 文件添加：`live-prod-ap-northeast-1.twitch.tv 你的电脑IP`
3. 在 PS5 的 Twitch IRC 设置中配置：
   - 服务器：你的电脑IP
   - 端口：6667
   - 频道名：`#你的Twitch频道名`

## 🔍 常见问题

### PS5 无法连接 IRC 服务
- 确保防火墙允许 6667 端口
- 确保在 PS5 上填写的 IP 地址是正确的电脑内网 IP
- 检查是否配置了 DNS 支持

### 弹幕不显示
- 确保已登录 B站 账号（在 Web 界面扫码登录）
- 确保输入的直播间 ID 正确
- 检查 Web 界面中的连接状态

### RTMP 推流状态不显示
- 确保 `monitor_rtmp.py` 正在运行
- 使用 `start_all.py` 启动全部服务
- 检查 nginx-rtmp 服务器是否正常运行
